About The Tool

Memory Watch is a program to display variables stored in emulator's memory.

Though this small program is not almighty, it may make your TAS life happier.

How To Use

Watch Variables

  1. Run the target emulator and open a game
  2. Run Memory Watch for the target emulator
  3. Edit and reload watch settings, if needed (see Edit Settings)

Edit Settings

  1. Open the config file (press Enter/Space key in watch window)
  2. Edit the config file (see Syntax)
  3. Reload settings (press F5 key in watch window)

Functions

Variables

[ROM Name / ROM Id]

Address1,Type1,Name1
Address2,Type2,Name2
----
Address3,Type3,Name3
ROM Name / ROM Id
String for identifying ROM, which written in ROM header (example: "KIRBY SUPER DELUXE" or "AKFE")
Address
Memory address of a variable (example: "7E149F", "7008EF", or "ARAM:0051", the address format depends on the emulator)
Type
Size and type of a variable, "[1-4][s|u|x|b](*)" (example: "1u", "2s*", or "4x") (* specifies that the variable is big-endian)
Name
Name of a variable (example: "X Velocity" or "Y Position")
----
Separator
